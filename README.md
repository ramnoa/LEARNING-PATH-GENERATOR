🎓 Gemini Learning Path Architect

An AI-powered learning roadmap generator that creates personalized, structured learning paths based on a student's goals, experience level, available study time, and preferred learning pace.

Built using Google's Gemini API and Google Search integration, the system automatically generates comprehensive study plans complete with free learning resources, schedules, projects, assessments, and progression milestones.

---

## 🚀 Overview

Gemini Learning Path Architect acts as an intelligent academic mentor that helps learners navigate complex subjects efficiently.

Instead of manually searching for courses, tutorials, and study materials, users simply provide their learning goal and timeframe, and the system generates a complete roadmap tailored to their needs.

The generated roadmap includes:

* Structured learning phases
* Weekly or daily study schedules
* Free learning resources
* Practice exercises
* Project recommendations
* Skill assessment checkpoints
* Advanced learning pathways

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/learning-path-architect.git
cd learning-path-architect
```

### Install Dependencies

```bash
pip install google-genai
```

### Configure API Key

Linux/Mac:

```bash
export GEMINI_API_KEY="YOUR_API_KEY"
```

Windows PowerShell:

```powershell
$env:GEMINI_API_KEY="YOUR_API_KEY"
```

### Run the Application

```bash
python main.py
```
